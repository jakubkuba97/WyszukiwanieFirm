
This program was made specifically to help a phone company's representative find corporations in his city, which might be willing to sign a contract.
It will search various informations about all the registered companies in the city of Poznan and save them to an excel file.

Data processed is publicly available and does not require any additional permissions in order to be accessed.
Software was originally intended for personal use only and was created to solidify the knowledge about Python skills such as:
	- working with Excel files
	- customizing and editing large amounts of data
	- file management
	- web searching and working with frameworks
	- logical problem solving
	- automating daily tasks
Note that the program was intended to be expanded upon later, hence it might be somewhat unoptimised in certain areas
This software was not tested on systems other than windows
The latest version of this product was finished on 25.05.2019
